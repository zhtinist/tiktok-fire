# 抖音自动续火花插件(Chrome / Edge)

每天首次打开浏览器时,自动在**后台静默**给所有正在**与你续火花的好友**发送一条固定消息
(默认 `1`),把火花续上。全程使用当前浏览器已登录的抖音账号,不需要填任何 cookie。

- ✅ 后台静默运行:打开一个不抢焦点的后台标签页,操作完成后自动关闭,**不弹窗、不打断你**
- ✅ 自动识别正在续火花的好友,逐个发送消息
- ✅ 完成后一条系统通知汇报结果
- ✅ 同时支持 Chrome 与 Edge(Manifest V3,代码通用)

## 安装

### 第 1 步:下载

👉 **[点此下载最新版 tiktok-fire.zip](https://github.com/zhtinist/tiktok-fire/releases/latest/download/tiktok-fire.zip)**

下载后**解压**,得到一个 `tiktok-fire` 文件夹(记住它的位置)。

> 为什么不用 `.crx`?浏览器出于安全会**拦截非商店来源的 `.crx`**,双击装不上。
> 所以统一用下面的"加载已解压的扩展程序"方式,加载上面解压出来的文件夹。

### 第 2 步:加载扩展

**Chrome 用户:**
1. 地址栏输入 `chrome://extensions` 回车
2. 打开右上角「**开发者模式**」
3. 点「**加载已解压的扩展程序**」→ 选择第 1 步解压出来的 `tiktok-fire` 文件夹

**Edge 用户(微软 Edge):**
Edge 与 Chrome 同为 Chromium 内核,插件完全通用:
1. 地址栏输入 `edge://extensions` 回车(或右上角 `···` → 扩展 → 管理扩展)
2. 打开左下角「**开发人员模式**」开关
3. 点「**加载解压缩的扩展**」→ 选择第 1 步解压出来的 `tiktok-fire` 文件夹
4. (可选)点地址栏右侧「扩展」拼图图标 → 把"抖音自动续火花"旁的眼睛点亮,固定到工具栏
5. 若系统没弹通知:macOS「系统设置 → 通知 → Microsoft Edge」打开允许通知

### 第 3 步:登录并验证
1. 在该浏览器里登录抖音网页版:https://www.douyin.com/chat
2. 点工具栏的插件图标 → 设置面板 →「**立即测试运行**」
3. 成功后会弹通知「今日(x月x日)已与 N 位用户续火花:…」,即安装完成 ✅

## 使用

1. 点击工具栏的插件图标 → 打开设置面板,可修改要发送的消息内容、开关、通知等。
2. **首次建议点一次「立即测试运行」**验证:它会静默跑一遍,成功后弹出通知
   「今日(x月x日)已与 N 位用户续火花:…」。
3. 之后**每天第一次启动浏览器**时自动执行;若你长期不关浏览器,还有每小时的兜底检查——
   跨过 0 点后当天第一次检查会自动再跑一次(每天只跑一次)。

## 完成通知

每次结束弹出系统通知,格式:

> 今日(x月x日)已与 N 位用户续火花:用户名1、用户名2……

最多显示 5 个用户名,超过以 `...` 结尾。可在设置面板关闭。

## 工作原理与限制

- 插件在后台标签页打开 `douyin.com/chat`,由内容脚本用真实鼠标/键盘事件模拟点击会话、
  输入并发送,完成后关闭标签页。之所以要开这个后台标签页,是因为抖音的登录态与请求签名
  都由其页面 JS 生成,无法在纯后台伪造请求——这已是"无感"的现实极限。
- **抖音改版可能导致失效**:若哪天不再发送成功,多半是抖音改了页面结构。可打开
  `content.js` 顶部的 `CONFIG` 更新选择器(会话项 `conversationItem`、火花识别
  `flameKeywords`、输入框 `chatInput`、发送按钮 `sendButton`)。
- 每天只执行一次(按本地日期去重,存于 `chrome.storage`)。

## 调试

如需排查,把 `background.js` 顶部的 `const DEBUG = false` 改为 `true`,重新加载扩展:
- 每次重载/启动都会运行,并打开**可见**标签页(方便观察)、不自动关闭;
- 每次运行把详细日志导出到「下载目录/`tiktok-fire-log.txt`」;
- 设置面板底部也有「运行日志」面板,可查看/复制/导出。

排查完记得把 `DEBUG` 改回 `false` 恢复静默模式。
