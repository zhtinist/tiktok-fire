# 抖音自动续火花 Chrome 插件

每天首次打开 Chrome 时,自动在后台给所有**有火花的好友**发送一条固定消息(默认 `1`),
续上火花。全程使用当前浏览器已登录的抖音账号,不需要额外填 cookie。

## 安装

同时支持 **Chrome** 和 **Edge**(两者都是 Chromium 内核,代码通用)。

**Chrome:**
1. 地址栏输入 `chrome://extensions`
2. 右上角打开「开发者模式」
3. 点「加载已解压的扩展程序」,选择本文件夹 `tiktok-fire`

**Edge:**
1. 地址栏输入 `edge://extensions`
2. 左下角打开「开发人员模式」
3. 点「加载解压缩的扩展」,选择本文件夹 `tiktok-fire`

安装后确保已在浏览器里登录抖音网页版 (https://www.douyin.com)。

## 完成通知

每次续火花结束后会弹出系统通知,格式为:

> 今日(x月x日)已与 x 位用户续火花:用户名1、用户名2……

最多显示 5 个用户名,超过则以 `...` 结尾。可在设置面板关闭。

## 使用

- 点击工具栏里的插件图标 → 打开设置面板,可修改要发送的消息、开关、间隔。
- **建议先点「立即测试运行」验证一次**,观察是否成功发送。
- 之后每天第一次启动 Chrome,插件会自动执行(靠 `onStartup`);
  如果你常年不关 Chrome,插件还有一个每小时的定时兜底检查,每天只会跑一次。

## 工作原理与限制(务必了解)

- 插件在**后台标签页**(不抢焦点)打开抖音,由内容脚本模拟点击会话、输入消息、发送,
  完成后自动关闭标签页。这是"无感"的现实极限——因为抖音的登录态和请求签名都由其页面
  JS 生成,无法在纯后台伪造请求。
- **选择器可能需要你自己微调**:抖音 DOM 经常改版。若测试运行时「命中火花会话」为 0
  或发送失败,请打开抖音私信页 → F12 → 对照 `content.js` 顶部的 `CONFIG`,更新其中的:
  - `conversationItem`:单个会话项的选择器
  - `flameKeywords`:判断火花的关键词(默认找 🔥 / "火花")
  - `chatInput`:聊天输入框
  - `sendButton` / `sendButtonText`:发送按钮
- 每天只会执行一次(以本地日期去重,记录在 `chrome.storage`)。

## 调试

- 后台日志:`chrome://extensions` → 本插件 →「Service Worker」→ 查看控制台。
- 内容脚本日志:在抖音标签页里按 F12 看 Console。
